# Bash Automation Lab

## Користувач
sc

## Базовий каталог
/opt

## Вміст репозиторію
- Bash-скрипти для автоматизації задач
- Налаштування cron
- systemd service та timer
- Конфігурація logrotate

## Структура проєкту
- `scripts/` — bash-скрипти
- `systemd/` — systemd service та timer
- `logrotate/` — конфігурації logrotate
