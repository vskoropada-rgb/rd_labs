# Lab №4 — Virtualization / VirtualBox

## Мета роботи
Ознайомитися з принципами віртуалізації, навчитися створювати
та налаштовувати віртуальні машини у VirtualBox, виконувати
інсталяцію операційної системи, працювати зі знімками (snapshots)
та змінювати апаратні параметри VM.

---

## Тестове середовище

- Хостова ОС: macOS
- Гіпервізор: Oracle VirtualBox
- Гостьова ОС: Ubuntu 24.04 LTS

---

## Завдання 1. Встановлення VirtualBox

Було завантажено та встановлено останню версію VirtualBox
з офіційного сайту:
https://www.virtualbox.org/

Після встановлення програма успішно запустилась та була готова до роботи.

---

## Завдання 2. Створення нової віртуальної машини

У VirtualBox було створено нову віртуальну машину з такими параметрами:

- Назва: `TestVM_ubuntu24_LTS`
- Тип: `Linux`
- Версія: `Ubuntu (64-bit)`
- Оперативна пам’ять (RAM): `2 ГБ`
- Віртуальний диск: `20 ГБ`, формат `VDI`

Початковий віртуальний жорсткий диск було додано під час створення VM.

---

## Завдання 3. Налаштування віртуальної машини

Перед запуском VM було виконано додаткові налаштування:

- CPU: встановлено `2 ядра`
- Увімкнено опцію **Enable EFI**
- Налаштовано мережу:
  - Adapter 1: `NAT` — для доступу до інтернету
  - Adapter 2: `Host-only Network` — для локальної мережі

## Завдання 4. Інсталяція операційної системи

1. Завантажено ISO-образ Ubuntu LTS
2. ISO додано у:
   - Settings → Storage → Optical Drive
3. Виконано стандартну установку Ubuntu на створений диск

Після завершення інсталяції система коректно завантажується та працює.

## Завдання 7. Зміна параметрів через GUI VirtualBox

Збільшення розміру диска
	1.	Віртуальну машину було вимкнено
	2.	У VirtualBox:
	•	Settings → Storage → Disk → Resize
	•	Розмір диска збільшено з 20 ГБ до 25 ГБ

Після цього у гостьовій ОС виконано розширення LVM: sudo pvresize /dev/sda3
Команда повідомляє LVM про зміну розміру фізичного диска.
	sudo lvextend -l +100%FREE /dev/mapper/ubuntu--vg-ubuntu--lv
Команда розширює логічний том, використовуючи весь доступний вільний простір.
	sudo resize2fs /dev/mapper/ubuntu--vg-ubuntu--lv
Команда розширює файлову систему для використання нового простору.

Зміна CPU та RAM

Через GUI VirtualBox було змінено апаратні параметри VM:
	•	CPU: збільшено до 4 ядер
	•	RAM: збільшено до 4 ГБ
